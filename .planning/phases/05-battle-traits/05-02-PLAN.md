**B. Create `src/traits/TraitSystem.ts`:**

This class is instantiated once per match in BattleScene, subscribes to EventBus events for the active trait's mechanic effects, and provides stat trait application as a static helper.

```typescript
import { EventBus, Events } from '../systems/EventBus';
import { Hero } from '../entities/Hero';
import { BaseEntity } from '../entities/BaseEntity';
import { TraitDef, BuffType, ActiveBuff, Team } from '../types';
import { getTraitById } from './traitData';

export class TraitSystem {
  private activeTrait: TraitDef;
  private heroes: Hero[];
  private handlerRefs: { event: string; handler: Function }[] = [];
  private scene: any; // BattleScene reference for finding team heroes

  // For sudden_valor rule_change: track first kill per minute
  private lastMinuteKillGranted = -1;

  constructor(traitDef: TraitDef, heroes: Hero[], scene: any) { ... }
  destroy(): void { ... }
  getActiveTrait(): TraitDef { return this.activeTrait; }

  // Static helpers for MatchOrchestrator
  static selectTrait(heroPassiveIds: string[]): TraitDef { ... }
  static applyStatMods(stats: any, trait: TraitDef): any { ... }
}
```

**Constructor logic:**
1. Store activeTrait, heroes, scene.
2. Subscribe mechanic event handlers based on trait's effect fields:
   - If `activeTrait.onHitEffect` exists: subscribe to `Events.HERO_HIT` with `this.handleOnHit`
   - If `activeTrait.onKillEffect` exists: subscribe to `Events.HERO_KILLED` with `this.handleOnKill`
   - If `activeTrait.onDamageTakenEffect` exists: subscribe to `Events.DAMAGE_TAKEN` with `this.handleOnDamageTaken`
3. Store each subscription in `handlerRefs` for cleanup.
4. Use arrow function wrappers stored in variables (same pattern as Hero.ts passiveHandlerRef):
   ```typescript
   if (traitDef.onHitEffect) {
     const handler = (payload: any) => this.handleOnHit(payload);
     EventBus.on(Events.HERO_HIT, handler, this);
     this.handlerRefs.push({ event: Events.HERO_HIT, handler });
   }
   ```

**handleOnHit(payload: { attacker: any, victim: any, damage: number }):**
- Guard: `if (!this.activeTrait.onHitEffect) return;`
- Guard: `if (!payload.attacker?.isAlive) return;` (attacker must be alive)
- If `healPercentOfDamage` > 0: calculate heal = `Math.floor(payload.damage * healPercentOfDamage)`. Call `payload.attacker.heal(heal)` -- lifesteal works on all targets (including boss) per research.
- If `dotChance` > 0 AND `Math.random() < dotChance`: check `payload.victim?.entityType === 'hero'` (DoT buffs only on heroes). If hero, create an ActiveBuff with `type: BuffType.DOT, value: dotDamage, duration: dotDuration, remaining: dotDuration, tickInterval: 1, tickTimer: 0, sourceId: 'trait_spell_burn'`. Call `payload.victim.addBuff(buff)`.

**handleOnKill(payload: { victim: BaseEntity, killerId?: string }):**
- Guard: `if (!this.activeTrait.onKillEffect) return;`
- Guard: `if (payload.victim.entityType !== 'hero') return;` (only hero kills trigger this)
- Guard: `if (payload.victim.isAlive) return;` (revival token was used -- victim is alive again, skip. This works because TraitSystem subscribes AFTER BattleScene's HERO_KILLED handler per EventEmitter3 subscription ordering. If a revival token was consumed, BattleScene already set victim.isAlive = true before this handler runs.)
- Find killer: `const killer = this.heroes.find(h => h.getUniqueId() === payload.killerId);`
- Guard: `if (!killer || !killer.isAlive) return;`
- If `damageBuff` > 0: apply STAT_BUFF to killer: `killer.addBuff({ type: BuffType.STAT_BUFF, value: damageBuff, duration: buffDuration, remaining: buffDuration, sourceId: 'trait_executioner' })`
- If `teamDamageBuff` > 0 (sudden_valor rule_change):
  - Calculate current minute: `const currentMinute = Math.floor((scene.matchStateMachine?.getTimeRemaining() ?? 300) / 60);` Wait -- time remaining counts down. Calculate elapsed minutes: `const elapsed = 300 - (scene.matchStateMachine?.getTimeRemaining() ?? 300); const currentMinute = Math.floor(elapsed / 60);`
  - If `this.lastMinuteKillGranted < currentMinute`: this is the first kill this minute.
    - Set `this.lastMinuteKillGranted = currentMinute`
    - Find all alive allies of the killer's team.
    - Apply STAT_BUFF to each: `{ type: BuffType.STAT_BUFF, value: teamDamageBuff, duration: teamBuffDuration, remaining: teamBuffDuration, sourceId: 'trait_sudden_valor' }`

**handleOnDamageTaken(payload: { victim: any, damage: number, sourceId?: string }):**
NOTE: The actual DAMAGE_TAKEN event emits `{ victim, sourceId, damage }` from Hero.takeDamage(). Use `payload.victim` (NOT `payload.entity`).
- Guard: `if (!this.activeTrait.onDamageTakenEffect) return;`
- Guard: `if (!payload.victim?.isAlive) return;`
- Guard: `if (payload.victim.entityType !== 'hero') return;` (only heroes reflect damage)
- If `reflectPercent` > 0: calculate reflected = `Math.floor(payload.damage * reflectPercent)`. Find source: `const source = this.heroes.find(h => h.getUniqueId() === payload.sourceId);` If source exists and is alive AND is a hero: `source.takeDamage(reflected, payload.victim.getUniqueId())`. This prevents reflect from hitting bosses/towers.

**destroy():**
```typescript
for (const ref of this.handlerRefs) {
  EventBus.off(ref.event, ref.handler, this);
}
this.handlerRefs = [];
```

**Static selectTrait(heroPassiveIds: string[]): TraitDef:**
- Import TRAITS from traitData.
- Filter out traits whose `incompatiblePassives` array has any overlap with `heroPassiveIds`.
- From the remaining eligible traits, pick one randomly with `Math.floor(Math.random() * eligible.length)`.
- If no eligible traits (edge case), return the first stat trait as fallback.

**Static applyStatMods(stats: any, trait: TraitDef): typeof stats:**
- Return a new object with trait stat modifiers applied additively:
  ```typescript
  return {
    ...stats,
    maxHP: stats.maxHP + (trait.hpMod ?? 0),
    damage: stats.damage + (trait.damageMod ?? 0),
    armor: stats.armor + (trait.armorMod ?? 0),
    moveSpeed: stats.moveSpeed + (trait.moveSpeedMod ?? 0),
    maxMana: stats.maxMana + (trait.manaMod ?? 0),
  };
  ```
- Apply caps: `maxHP: Math.max(100, ...)` to prevent HP going below 100 from glass_cannon on low-HP heroes.
