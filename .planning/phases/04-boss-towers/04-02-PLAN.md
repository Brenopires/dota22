  <action>
Create `src/entities/TowerEntity.ts` extending BaseEntity. This is a STATIC entity -- it never moves.

The tower constants (TOWER_MAX_HP, TOWER_ATTACK_DAMAGE, etc.) and events (TOWER_DESTROYED, TOWER_DISABLED, etc.) were added in plan 04-01 and are available for import.

**Class structure:**
- `readonly entityType = 'tower' as const`
- Private fields: `attackTimer = 0`, `lastDamagedTime = 0`, `disabled = false`, `disableTimer = 0`, `attackRadius: number`, `attackDamage: number`
- Private visual refs: `rangeIndicator`, `baseGraphics`, `labelText`, `glowCircle`

**Constructor (scene, x, y, team: Team):**
- Call `super(scene, x, y, TOWER_MAX_HP, team)` -- team is A or B (NOT neutral).
- Store `attackRadius = TOWER_ATTACK_RADIUS`, `attackDamage = TOWER_ATTACK_DAMAGE`.
- Create visuals as Container children (following Hero pattern):
  - Attack range indicator: faint circle (TOWER_ATTACK_RADIUS) with team-colored stroke (alpha 0.1), no fill. This shows the AoE range.
  - Base structure: Graphics object -- fillRoundedRect centered on (0,0) with size TOWER_RADIUS*2, color 0x888888. Team-colored stroke border (3px, A=0x00aaff, B=0xff4444, alpha 0.8).
  - Team glow: circle (TOWER_RADIUS + 5) with team-colored stroke (2px, alpha 0.3) and very faint fill (alpha 0.05).
  - Label: text "T" centered, 14px monospace bold white.
  - Health bar: Use the existing HealthBar class from `../entities/HealthBar` -- instantiate and add its game objects to the container, same as Hero does.
  - Pulsing glow tween on the glow circle (alpha 0.2 to 0.4, duration 2000, yoyo, repeat -1).
- Add to scene: `scene.add.existing(this)`, `scene.physics.add.existing(this, false)` (NOT static -- BaseEntity expects dynamic body).
- Configure physics body: `setImmovable(true)`, `setCircle(TOWER_RADIUS, -TOWER_RADIUS, -TOWER_RADIUS)`, `setVelocity(0, 0)`, `body.moves = false` (never moves), `setCollideWorldBounds(true)`.

**Public methods:**

`updateTower(dt: number, enemies: BaseEntity[]): void`
- If `!this.isAlive` return immediately.
- If disabled: decrement disableTimer by dt. If disableTimer <= 0, call `this.enable()`. Return (no attack, no regen while disabled).
- Call `this.updateBuffs(dt)`.
- **Attack logic:** Decrement attackTimer by dt. If attackTimer <= 0:
  - Filter enemies to find alive entities within TOWER_ATTACK_RADIUS using `Phaser.Math.Distance.Between(this.x, this.y, e.x, e.y)`.
  - IMPORTANT: Only target entities that are NOT Team.NEUTRAL (towers do NOT attack the boss). Check `e.team !== Team.NEUTRAL`.
  - If targets found, pick the closest one, call `target.takeDamage(this.attackDamage, this.getUniqueId())`, reset attackTimer = TOWER_ATTACK_INTERVAL, call `this.showAttackVFX(target)`.
- **Regen logic:** Calculate timeSinceDamage = `this.scene.time.now - this.lastDamagedTime`. If timeSinceDamage > TOWER_REGEN_DELAY AND currentHP < maxHP: call `this.heal(TOWER_REGEN_RATE * dt)`.
- Update health bar.

`disable(duration: number): void`
- Set `this.disabled = true`, `this.disableTimer = duration`.
- Emit `Events.TOWER_DISABLED` with `{ tower: this, duration }`.
- Visual feedback: set alpha to 0.4 on the base graphics, add a grey tint overlay.

`isDisabled(): boolean` -- Returns `this.disabled`.

**Private methods:**

`enable(): void`
- Set `this.disabled = false`, `this.disableTimer = 0`.
- Emit `Events.TOWER_ENABLED` with `{ tower: this }`.
- Visual feedback: restore alpha to 1.0, remove grey overlay.

`showAttackVFX(target: BaseEntity): void`
- Draw a line from tower center to target position using Graphics (team-colored, 2px, alpha 0.6).
- Tween the line to fade out over 200ms then destroy.
- Show a small impact flash circle at target position (same pattern as CombatSystem.showMeleeSlash impact flash).

`getUniqueId(): string` -- Returns `'tower_' + this.team`.

`getArmor(): number` -- Returns 10 (towers have flat armor, higher than heroes).

**Override takeDamage:**
- Set `this.lastDamagedTime = this.scene.time.now` (track for regen delay).
- Call and return `super.takeDamage(rawDamage, sourceId)`.
- If finalDamage > 0, emit `Events.TOWER_DAMAGED` with `{ tower: this, damage: finalDamage }`.

**Override die(killerId):**
- CRITICAL: Do NOT call super.die() -- it emits HERO_KILLED.
- Duplicate the idempotent guard: `if (!this.isAlive) return;`
- Set `this.isAlive = false`, disable physics body.
- Emit `Events.TOWER_DESTROYED` with `{ tower: this, destroyedTeam: this.team, killerId }`.
- Call `this.onDeath(killerId)`.

**Protected onDeath(killerId):**
- Large explosion VFX via vfxManager if available.
- Tween: scale down + fade + red flash over 1000ms.
- Show "TOWER DESTROYED!" text in large font at tower position.

**Do NOT:**
- Do NOT make the tower target the boss (Team.NEUTRAL entities excluded from targeting).
- Do NOT emit HERO_KILLED from die().
- Do NOT give the tower movement -- it is immovable.
