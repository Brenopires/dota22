  <files>src/ui/BossHealthBar.ts</files>
  <action>
Create `src/ui/BossHealthBar.ts` -- a screen-space health bar fixed at the top of the viewport, centered horizontally.

**Positioning:** Below the timer (timer is at y=28 with radius 32, so bottom of timer is ~60). Boss bar at y=80, centered.

**Constants (local to this file):**
- BAR_WIDTH = 400
- BAR_HEIGHT = 16
- BAR_Y = 80

**Constructor (scene: Phaser.Scene):**

1. Create main Graphics object for HP fill. `setScrollFactor(0).setDepth(200)`.

2. Create phase threshold markers (separate Graphics, depth 201, scrollFactor 0):
   - Yellow vertical line at 60% of BAR_WIDTH (ENRAGED threshold): from (barX + BAR_WIDTH * 0.6, BAR_Y - 2) to (barX + BAR_WIDTH * 0.6, BAR_Y + BAR_HEIGHT + 2). lineStyle(2, 0xffff00, 0.6).
   - Red vertical line at 25% of BAR_WIDTH (DYING threshold): lineStyle(2, 0xff0000, 0.6).
   - barX = (GAME_WIDTH - BAR_WIDTH) / 2.

3. Create boss name text: "ANCIENT GUARDIAN", centered above the bar at (GAME_WIDTH/2, BAR_Y - 14). fontSize 12px, color #ff6666, monospace bold. setScrollFactor(0), depth 201.

4. Create HP text: centered on bar (GAME_WIDTH/2, BAR_Y + BAR_HEIGHT/2). fontSize 10px, white, monospace. setScrollFactor(0), depth 202.

**Public update(currentHP: number, maxHP: number, phase: BossPhase): void**

1. Clear the graphics object.
2. Calculate barX = (GAME_WIDTH - BAR_WIDTH) / 2, hpRatio = Math.max(0, currentHP / maxHP).
3. Draw background: fillStyle(0x333333, 0.8), fillRoundedRect(barX, BAR_Y, BAR_WIDTH, BAR_HEIGHT, 4).
4. Draw HP fill:
   - Color based on phase: NORMAL -> 0xff4444 (red), ENRAGED -> 0xff8800 (orange), DYING -> 0xff0000 (bright red).
   - hpWidth = BAR_WIDTH * hpRatio.
   - If hpWidth > 0: fillStyle(phaseColor, 1), fillRoundedRect(barX, BAR_Y, hpWidth, BAR_HEIGHT, 4).
   - Add highlight on top half: fillStyle(0xffffff, 0.1), fillRoundedRect(barX, BAR_Y, hpWidth, BAR_HEIGHT / 2, {tl:4, tr:4, bl:0, br:0}).
5. Draw border: lineStyle(2, 0x888888, 0.6), strokeRoundedRect(barX, BAR_Y, BAR_WIDTH, BAR_HEIGHT, 4).
6. Update HP text: `${Math.ceil(currentHP)} / ${maxHP}`.

**Public setVisible(visible: boolean): void**
- Set visibility on all graphics, texts. Used to hide when boss is dead.

**Public destroy(): void**
- Destroy all Phaser objects (graphics, texts).

   ```
   const battleScene = this.scene as any;
   if (battleScene.boss) {
     const boss = battleScene.boss;
     if (boss.isAlive) {
       this.bossHealthBar.setVisible(true);
       this.bossHealthBar.update(boss.currentHP, boss.maxHP, boss.phase);
     } else {
       this.bossHealthBar.setVisible(false);
     }
   }
   ```

2. **Tower status updates:**
   Update each tower's mini health bar. Access towers via `battleScene.towerA` and `battleScene.towerB`.

   For each tower (A and B):
   ```
   const tower = battleScene.towerA; // or towerB
   const g = this.towerAGraphics; // or towerBGraphics
   g.clear();
   if (tower && tower.isAlive) {
     const hpRatio = tower.currentHP / tower.maxHP;
     const barX = [position]; // calculated based on tower side
     const barWidth = 60;
     const barHeight = 8;
     // Background
     g.fillStyle(0x333333, 0.8);
     g.fillRoundedRect(barX, 54, barWidth, barHeight, 2);
     // HP fill
     const teamColor = [0x00aaff or 0xff4444];
     const fillWidth = barWidth * hpRatio;
     if (fillWidth > 0) {
       g.fillStyle(teamColor, 1);
       g.fillRoundedRect(barX, 54, fillWidth, barHeight, 2);
     }
     // If disabled, show "DISABLED" text overlay or grey fill
     if (tower.isDisabled()) {
       g.fillStyle(0x666666, 0.5);
       g.fillRoundedRect(barX, 54, barWidth, barHeight, 2);
       this.towerAText.setText('T-A [OFF]'); // or T-B [OFF]
       this.towerAText.setColor('#666666');
     } else {
       this.towerAText.setText('T-A');
       this.towerAText.setColor('#00aaff');
     }
   } else if (tower && !tower.isAlive) {
     // Tower destroyed -- show "DESTROYED" indicator
     this.towerAText.setText('T-A [X]');
     this.towerAText.setColor('#444444');
   }
   ```
