  <action>
Create `src/systems/BossAISystem.ts` -- a standalone system that drives boss behavior. NOT extending AIController (boss has no abilities, mana, or hero target selection logic).

**Constructor (boss: BossEntity, scene: Phaser.Scene, homeX: number, homeY: number):**
- Store boss ref, scene ref, homePosition = { x: homeX, y: homeY }.
- Private fields: `aggroTarget: BaseEntity | null = null`, `stickyTargetTimer = 0` (prevents ping-pong aggro switching).

**Public update(dt: number, heroes: Hero[]): void**
- If `!this.boss.isAlive` return.
- Call `this.boss.updateBoss(dt)`.
- Decrement stickyTargetTimer by dt.

**Step 1: Find targets in aggro radius.**
- Filter heroes to find alive heroes within BOSS_AGGRO_RADIUS of boss position.
- Use `Phaser.Math.Distance.Between(this.boss.x, this.boss.y, h.x, h.y)`.
- Boss attacks ALL teams (neutral, attacks everyone) -- no team filter needed.

**Step 2: Aggro management.**
- If aggroTarget exists and is alive and within BOSS_LEASH_RADIUS: keep current target (sticky).
- If aggroTarget is null or dead or stickyTargetTimer <= 0: pick new target = closest hero in aggro range. Set stickyTargetTimer = 2.0 (seconds, prevents rapid switching).
- If no heroes in aggro range: clear aggroTarget, call `this.returnToHome(dt)`, return.

**Step 3: Move toward target.**
- Calculate angle to aggroTarget.
- If distance > BOSS_ATTACK_RANGE: move boss toward target at BOSS_MOVE_SPEED via body.setVelocity.
- If distance <= BOSS_ATTACK_RANGE: stop movement, execute attack.

**Step 4: Execute phase-appropriate attack.**
- Check if `boss.attackTimer <= 0` (boss is ready to attack, attackTimer is decremented in boss.updateBoss).
- Reset boss attackTimer to `boss.getAttackInterval()`.
- Switch on `boss.phase`:

  **NORMAL phase: Single-target melee.**
  - Deal boss.getAttackDamage() to aggroTarget via `aggroTarget.takeDamage(damage, boss.getUniqueId())`.
  - Show melee VFX: red slash arc from boss to target (similar to CombatSystem.showMeleeSlash pattern but larger, red/orange colored).

  **ENRAGED phase: Melee + AoE ground slam.**
  - Deal boss.getAttackDamage() to aggroTarget (primary target, full damage).
  - AoE: find all heroes within 120px of boss. Deal 50% of getAttackDamage() to each (excluding primary target to avoid double-hit).
  - Show AoE VFX: expanding orange ring from boss position (similar to AreaEffect visual pattern).
  - Optional screen shake on AoE hit via vfxManager.

  **DYING phase: Rapid attacks + larger AoE.**
  - Deal boss.getAttackDamage() * 1.2 to aggroTarget (desperate strikes).
  - AoE: find all heroes within 150px of boss. Deal 60% of getAttackDamage() to each.
  - Show VFX: red pulsing ring, more intense screen shake.
  - Attack interval is fastest (BOSS_DYING_ATTACK_INTERVAL = 0.7s).

**Private returnToHome(dt: number): void**
- If boss is within 10px of homePosition: set velocity 0, return.
- Otherwise: move boss toward homePosition at BOSS_MOVE_SPEED.
- IMPORTANT: Do NOT heal boss when returning. Boss only heals via explicit heal() calls (no leash-reset exploit).

**Anti-patterns to avoid:**
- Do NOT heal boss to full when returning to home (leash exploit prevention per research).
- Do NOT have boss attack towers (boss only attacks heroes per research recommendation).
- Boss targets ALL heroes regardless of team (it is neutral).

```typescript
tryAutoAttack(hero: Hero): void {
  if (!hero.isAlive || hero.isStunned()) return;
  if (hero.autoAttackTimer > 0) return;

  const battleScene = this.scene as any;
  const enemies = battleScene.getEnemies(hero.team) as Hero[];

  const range = hero.getAttackRange();
  let closest: Hero | BaseEntity | null = null;
  let closestDist = Infinity;

  // Check hero enemies (existing logic)
  if (enemies) {
    for (const enemy of enemies) {
      const dist = hero.distanceTo(enemy);
      if (dist <= range && dist < closestDist) {
        closestDist = dist;
        closest = enemy;
      }
    }
  }

  // ALSO check boss and enemy towers as valid targets
  const nonHeroTargets: BaseEntity[] = battleScene.getNonHeroTargets?.(hero.team) || [];
  for (const target of nonHeroTargets) {
    if (!target.isAlive) continue;
    const dist = Phaser.Math.Distance.Between(hero.x, hero.y, target.x, target.y);
    if (dist <= range && dist < closestDist) {
      closestDist = dist;
      closest = target;
    }
  }

  if (closest) {
    const damage = hero.getAttackDamage();
    // IMPORTANT: Pass hero.getUniqueId() as sourceId so boss/tower death
    // can trace back to the killer for rewards (killerId propagation).
    closest.takeDamage(damage, hero.getUniqueId());
    EventBus.emit(Events.HERO_HIT, {
      attacker: hero,
      victim: closest,
      damage,
    });
    hero.autoAttackTimer = AUTO_ATTACK_COOLDOWN / 1000;

    // VFX: check if target is a Hero for existing VFX methods,
    // otherwise show a generic attack VFX
    if (closest instanceof Hero) {
      if (hero.isRanged()) {
        this.showRangedAttack(hero, closest);
      } else {
        this.showMeleeSlash(hero, closest);
      }
    } else {
      // Generic attack VFX for boss/tower targets
      this.showGenericAttack(hero, closest);
    }
  }
}
```

```typescript
private showGenericAttack(attacker: Hero, target: BaseEntity): void {
  if (attacker.isRanged()) {
    // Ranged: small projectile dot flying to target
    const dot = this.scene.add.circle(attacker.x, attacker.y, 4, attacker.stats.color, 0.8);
    dot.setDepth(5);
    this.scene.tweens.add({
      targets: dot,
      x: target.x,
      y: target.y,
      duration: 100,
      ease: 'Linear',
      onComplete: () => {
        dot.destroy();
        const flash = this.scene.add.circle(target.x, target.y, 10, 0xffffff, 0.5);
        flash.setDepth(4);
        this.scene.tweens.add({
          targets: flash,
          alpha: 0, scaleX: 1.5, scaleY: 1.5,
          duration: 150,
          onComplete: () => flash.destroy(),
        });
      },
    });
  } else {
    // Melee: slash arc toward target
    const angle = Math.atan2(target.y - attacker.y, target.x - attacker.x);
    const g = this.scene.add.graphics();
    g.setDepth(6);
    g.lineStyle(3, attacker.stats.color, 0.8);
    const cx = attacker.x + Math.cos(angle) * 15;
    const cy = attacker.y + Math.sin(angle) * 15;
    g.beginPath();
    g.arc(cx, cy, 30, angle - 0.8, angle + 0.8, false);

  }
}
```
