Create BaseEntity as the abstract base class for all damageable game objects, then refactor Hero to extend it. This severs the direct Hero → BattleScene.onHeroKill() coupling that currently ends the match on every death.

Output: src/entities/BaseEntity.ts (new), src/entities/Hero.ts (refactored to extend BaseEntity).
</objective>

      const body = this.body as Phaser.Physics.Arcade.Body;
      body?.setVelocity(0, 0);
      body?.setEnable(false);
      body?.setCircle(0);

    }
