**Class: NeutralMob extends BaseEntity**

**Properties:**
- `readonly entityType = 'neutral_mob' as const`
- `campType: CampType` -- which camp type this mob guards
- `attackTimer = 0` -- public, like BossEntity.attackTimer
- `private homePosition: { x: number; y: number }` -- spawn origin for leash return
- `private baseMaxHP: number` -- for scaling (same pattern as BossEntity)
- `private baseDamage: number` -- for scaling
- `private minutesElapsed = 0` -- for scaling
- `private mobCircle: Phaser.GameObjects.Arc` -- visual reference for color
- `private mobLabel: Phaser.GameObjects.Text` -- camp type letter

**Constructor(scene, x, y, campType: CampType):**
- Call `super(scene, x, y, CAMP_MOB_HP, Team.NEUTRAL)`
- Store campType, homePosition = { x, y }, baseMaxHP = CAMP_MOB_HP, baseDamage = CAMP_MOB_DAMAGE
- **Visuals (Container children, following BossEntity pattern):**
  - Glow ring: `scene.add.circle(0, 0, CAMP_MOB_RADIUS + 4, campColor, 0.08)` with ADD blend + subtle pulse tween (alpha 0.05-0.15, scale 1.0-1.08, 1200ms yoyo)
  - Outer ring: `scene.add.circle(0, 0, CAMP_MOB_RADIUS + 6)` with setStrokeStyle(1.5, campColor) + pulse tween (alpha 0.3-0.7, 1500ms yoyo)
  - Main circle: `scene.add.circle(0, 0, CAMP_MOB_RADIUS, campColor)` -- stored as this.mobCircle
  - Label text: single letter centered (D/S/H/C for Damage/Shield/Haste/Cooldown) white, monospace bold 12px -- stored as this.mobLabel
- **Camp color map:**
  - CampType.DAMAGE = 0xFF4444 (red)
  - CampType.SHIELD = 0xCCCCCC (light grey/white)
  - CampType.HASTE = 0x00FFFF (cyan)
  - CampType.COOLDOWN = 0xAA44FF (purple)
- **Camp label map:**
  - CampType.DAMAGE = 'D', SHIELD = 'S', HASTE = 'H', COOLDOWN = 'C'
- `scene.add.existing(this)` + `scene.physics.add.existing(this)`
- Configure physics body: `body.setCircle(CAMP_MOB_RADIUS, -CAMP_MOB_RADIUS, -CAMP_MOB_RADIUS)`, collideWorldBounds(true), bounce(0), drag(400), maxVelocity(CAMP_MOB_MOVE_SPEED)

**Methods:**

`getUniqueId(): string` -- return `'camp_' + this.campType` (e.g., 'camp_damage')

`getAttackDamage(): number` -- return `Math.floor(this.baseDamage * (1 + this.minutesElapsed * CAMP_MOB_SCALING_PER_MINUTE))`

`getArmor(): number` -- return `CAMP_MOB_ARMOR`

`scalePower(minute: number): void` -- same pattern as BossEntity:
- this.minutesElapsed = minute
- const multiplier = 1 + (minute * CAMP_MOB_SCALING_PER_MINUTE)
- newMaxHP = Math.floor(baseMaxHP * multiplier), heal only delta, update maxHP

`updateMob(dt: number): void` -- tick buffs and attack timer:
- this.updateBuffs(dt)
- if (this.attackTimer > 0) this.attackTimer -= dt

`moveToward(targetX: number, targetY: number): void`:
- Set velocity toward target at CAMP_MOB_MOVE_SPEED using angle calculation

`stopMoving(): void`:
- body.setVelocity(0, 0)

`returnToHome(): void`:
- Calculate distance to homePosition
- If dist < 10: stop, full-heal (this.currentHP = this.maxHP), return
- Else: move toward homePosition at CAMP_MOB_MOVE_SPEED

`showMeleeVFX(target: BaseEntity): void`:
- Same pattern as BossAISystem.showMeleeVFX but smaller (radius 25, lineWidth 3, campColor):
  - Slash arc from mob toward target
  - Impact flash at target position (circle radius 10)

**override takeDamage(rawDamage, sourceId): number** -- show floating damage number:
- Call super.takeDamage, if finalDamage > 0 show damage number (same pattern as BossEntity.showBossDamageNumber but with camp color)

**protected die(killerId?: string): void** -- CRITICAL: does NOT call super.die():
- Idempotent guard: if (!this.isAlive) return
- this.isAlive = false
- Disable physics: body.setVelocity(0,0), body.setEnable(false), body.setCircle(0)
- Emit: EventBus.emit(Events.CAMP_CLEARED, { victim: this, killerId, campType: this.campType })
- Call this.onDeath(killerId)

**protected onDeath(killerId?: string): void**:
- VFX death explosion if vfxManager available (use camp color)
- Shrink + fade + rotate tween (400ms, Back.easeIn) -> setVisible(false)

**respawn(): void** -- called by NeutralCampSystem to bring mob back:
- this.isAlive = true
- this.currentHP = this.maxHP
- this.buffs = []
- this.shield = 0
- this.attackTimer = 0
- setPosition(homePosition.x, homePosition.y)
- setScale(1), setAlpha(0), setVisible(true), setAngle(0)
- Re-enable physics body: setEnable(true), setCircle(CAMP_MOB_RADIUS, -CAMP_MOB_RADIUS, -CAMP_MOB_RADIUS), collideWorldBounds(true), bounce(0), drag(400), maxVelocity(CAMP_MOB_MOVE_SPEED), setVelocity(0,0)
- Fade-in tween (alpha 0 to 1, 500ms, Power2)
