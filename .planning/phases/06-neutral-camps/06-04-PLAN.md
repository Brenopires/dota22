**Camp positions constant (module-level, not exported):**
```typescript
const CAMP_POSITIONS: Record<CampType, { x: number; y: number; name: string }> = {
  [CampType.DAMAGE]:   { x: 800,  y: 250,  name: 'North Camp' },
  [CampType.SHIELD]:   { x: 800,  y: 950,  name: 'South Camp' },
  [CampType.HASTE]:    { x: 500,  y: 600,  name: 'West Camp' },
  [CampType.COOLDOWN]: { x: 1100, y: 600,  name: 'East Camp' },
};
```

Note: West/East positions are x=500/x=1100 (not 400/1200) per research recommendation for breathing room from towers at x=250/x=1350.

**Class: NeutralCampSystem**

**Properties:**
- `private scene: Phaser.Scene`
- `private heroes: Hero[]`
- `private hud: HUD`
- `private mobs: Map<CampType, NeutralMob> = new Map()` -- the 4 camp mob instances
- `private respawnTimers: Map<CampType, Phaser.Time.TimerEvent> = new Map()` -- active respawn timers
- `private minutesElapsed = 0` -- for camp scaling

**Constructor(scene, heroes, hud):**
- Store refs
- Spawn all 4 camps: iterate CampType values, call this.spawnCamp(campType)
- Subscribe to CAMP_CLEARED: `EventBus.on(Events.CAMP_CLEARED, this.onCampCleared, this)`

**spawnCamp(campType: CampType): void**
- Get position from CAMP_POSITIONS[campType]
- If mob already exists in this.mobs for this campType, call mob.respawn() and emit CAMP_RESPAWNED
- Else create new NeutralMob: `const mob = new NeutralMob(this.scene, pos.x, pos.y, campType)`
  - If minutesElapsed > 0, call mob.scalePower(minutesElapsed) so respawned camps match current scaling
  - Store in this.mobs.set(campType, mob)
  - Emit: EventBus.emit(Events.CAMP_RESPAWNED, { campType })
- Add subtle zone label at camp position: faint text with camp name (e.g., "NORTH") at y-30 above camp, alpha 0.15, fontSize 10px, monospace. This helps players orient. Set depth 1 so it's behind entities.

**getAliveMobs(): NeutralMob[]**
- Return array of all alive mobs from this.mobs (filter by mob.isAlive)
- This is called by BattleScene.getNonHeroTargets()

**update(dt: number, heroes: Hero[]): void**
- For each mob in this.mobs.values(): if mob.isAlive, call this.updateCampAI(mob, dt, heroes)

**scaleCamps(minute: number): void**
- this.minutesElapsed = minute
- For each mob in this.mobs.values(): if mob.isAlive, call mob.scalePower(minute)

**private updateCampAI(mob: NeutralMob, dt: number, heroes: Hero[]): void**
Simplified BossAISystem pattern -- NO sticky target, NO phase transitions, NO AoE.

1. Call mob.updateMob(dt) -- ticks buffs + attack timer
2. Find closest alive hero within CAMP_MOB_AGGRO_RADIUS:
```typescript
let closest: Hero | null = null;
let closestDist = Infinity;
for (const h of heroes) {
  if (!h.isAlive) continue;
  const dist = Phaser.Math.Distance.Between(mob.x, mob.y, h.x, h.y);
  if (dist <= CAMP_MOB_AGGRO_RADIUS && dist < closestDist) {
    closest = h;
    closestDist = dist;
  }
}
```
3. If no hero in range: mob.returnToHome(), return
4. Check leash: if distance from mob to homePosition > CAMP_MOB_LEASH_RADIUS: mob.returnToHome(), return
5. If closestDist > CAMP_MOB_ATTACK_RANGE: mob.moveToward(closest.x, closest.y)
6. Else (in attack range):
   - mob.stopMoving()
   - if mob.attackTimer <= 0:
     - closest.takeDamage(mob.getAttackDamage(), mob.getUniqueId())
     - mob.attackTimer = CAMP_MOB_ATTACK_INTERVAL
     - mob.showMeleeVFX(closest)

**private onCampCleared({ victim, killerId, campType }: { victim: BaseEntity; killerId?: string; campType: CampType }): void**
1. Find killer hero: `const killer = this.heroes.find(h => h.getUniqueId() === killerId)`
   - If no killer found, return (edge case: killed by DoT with no source)
2. Get alive allies on killer's team: `const allies = this.heroes.filter(h => h.team === killer.team && h.isAlive)`
3. Create buff: `const buff = this.createCampBuff(campType)`
4. Apply buff to all alive allies: `for (const ally of allies) { ally.addBuff({ ...buff }); }`
5. Emit: EventBus.emit(Events.CAMP_BUFF_GRANTED, { campType, team: killer.team, duration: CAMP_BUFF_DURATION })
6. Kill feed: `this.hud.showKill('TEAM ' + killer.team, campType.toUpperCase() + ' CAMP')`
7. Schedule respawn: `this.scheduleCampRespawn(campType)`

**private createCampBuff(campType: CampType): ActiveBuff** (import ActiveBuff from types)
Return the appropriate buff object based on camp type:
- CampType.DAMAGE: { type: BuffType.STAT_BUFF, value: CAMP_BUFF_DAMAGE_VALUE, duration: CAMP_BUFF_DURATION, remaining: CAMP_BUFF_DURATION, sourceId: 'camp_damage' }
- CampType.SHIELD: { type: BuffType.SHIELD, value: CAMP_BUFF_SHIELD_VALUE, duration: CAMP_BUFF_DURATION, remaining: CAMP_BUFF_DURATION, sourceId: 'camp_shield' }
- CampType.HASTE: { type: BuffType.HASTE, value: CAMP_BUFF_HASTE_VALUE, duration: CAMP_BUFF_DURATION, remaining: CAMP_BUFF_DURATION, sourceId: 'camp_haste' }
- CampType.COOLDOWN: { type: BuffType.COOLDOWN_REDUCTION, value: CAMP_BUFF_COOLDOWN_VALUE, duration: CAMP_BUFF_DURATION, remaining: CAMP_BUFF_DURATION, sourceId: 'camp_cooldown' }

**private scheduleCampRespawn(campType: CampType): void**
Following BattleScene hero respawn timer pattern:
```typescript
const timer = this.scene.time.delayedCall(CAMP_RESPAWN_DELAY, () => {
  this.spawnCamp(campType);
  this.respawnTimers.delete(campType);
});
this.respawnTimers.set(campType, timer);
```

**destroy(): void** -- CRITICAL for match restart cleanup
1. Remove EventBus listener: `EventBus.off(Events.CAMP_CLEARED, this.onCampCleared, this)`
2. Clear all respawn timers:
```typescript
for (const timer of this.respawnTimers.values()) {
  this.scene.time.removeEvent(timer);
}
this.respawnTimers.clear();
```
3. Clear mobs map: `this.mobs.clear()`
