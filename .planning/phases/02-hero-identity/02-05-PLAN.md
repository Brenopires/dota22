**update() additions:**

In the `if (scene.player && scene.player.isAlive)` block, after the mana bar rendering, add XP bar rendering:

```typescript
// XP bar
const player = scene.player;
const xpThresholds = XP_THRESHOLDS;
const xpPrev = xpThresholds[player.level - 1] ?? 0;
const xpNext = xpThresholds[player.level] ?? xpThresholds[xpThresholds.length - 1];
const xpProgress = xpNext > xpPrev
  ? Math.min(1, (player.currentXP - xpPrev) / (xpNext - xpPrev))
  : 1; // at max level, show full bar

const xg = this.xpGraphics;
xg.clear();

// XP bar background
xg.fillStyle(0x111111, 1);
xg.fillRoundedRect(20, GAME_HEIGHT - 36, this.xpBarWidth, 6, 2);

// XP fill
const xpFillWidth = Math.max(0, this.xpBarWidth * xpProgress);
if (xpFillWidth > 0) {
  xg.fillStyle(0xFFD700, 1);
  xg.fillRoundedRect(20, GAME_HEIGHT - 36, xpFillWidth, 6, 2);
}

this.levelText.setText(`LV ${player.level}`);
```

Note on XP bar position: `GAME_HEIGHT - 36` places the 6px bar just below the 10px mana bar which ends around `GAME_HEIGHT - 40`. Adjust if there's visual overlap — the goal is a compact strip below mana.

Note on max level: If `player.level >= xpThresholds.length - 1`, the hero is at max level — show full gold bar. The while loop in gainXP() already stops there.

Do NOT add an EventBus listener for HERO_LEVELED_UP in HUD — the XP bar just reads `player.currentXP` and `player.level` directly each frame in update(), like the HP bar reads `player.currentHP`. No subscription needed.
