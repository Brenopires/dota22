## v1 Requirements — All Complete

### Match Flow

- [x] **FLOW-01**: Match has a 5-minute hard cap with visible countdown timer — Phase 1
- [x] **FLOW-02**: Heroes respawn after death with max 10-second respawn timer — Phase 1
- [x] **FLOW-03**: Match ends with score calculation: hero kill (1pt), boss kill (3pt), tower damage threshold (2pt), neutral control majority (1pt) — Phase 7
- [x] **FLOW-04a**: Match progresses through states: PRE_MATCH -> ACTIVE -> ENDED with forward-only transitions — Phase 1
- [x] **FLOW-04b**: Match state machine supports SUDDEN_DEATH state between ACTIVE and ENDED — Phase 7
- [x] **FLOW-05**: Tie at 5:00 triggers Sudden Death mode (no respawns, last team standing wins) — Phase 7
- [x] **FLOW-06**: Match can end early via tower destruction (instant win) or boss tier 3 kill (triggers Sudden Death) — Phase 7

### Heroes

- [x] **HERO-01**: 6-10 playable heroes with distinct playstyles covering different combat roles — Phase 2 (13 heroes shipped)
- [x] **HERO-02**: Each hero has 3 basic abilities (Q/W/E) + 1 ultimate (R) + 1 passive — Phase 2
- [x] **HERO-03**: Heroes gain XP from combat kills and objective interactions (no gold system) — Phase 2
- [x] **HERO-04**: Heroes level up during match with stat scaling (HP, damage, ability power) — Phase 2
- [x] **HERO-05**: Each match assigns a random Battle Trait modifier — Phase 5 (8 traits shipped)
- [x] **HERO-06**: Each hero receives a random Gem power-up at match start — Phase 5 (8 gems shipped)

### Asymmetric Teams

- [x] **ASYM-01**: System randomly assigns team sizes each match (any combination 1-5 per side) — Phase 3
- [x] **ASYM-02**: Solo/smaller team receives MMR-situational adaptive scaling — Phase 3
- [x] **ASYM-03**: AI teammates distribute targets to prevent focus-fire — Phase 3

### Boss System

- [x] **BOSS-01**: Central boss spawns immediately at match start and scales in power every minute — Phase 4
- [x] **BOSS-02**: First boss kill grants team-wide stat buff + 1 revival token — Phase 4
- [x] **BOSS-03**: Second boss kill grants permanent damage amplification and boss begins roaming — Phase 7
- [x] **BOSS-04**: Third boss kill triggers Sudden Death mode — Phase 7
- [x] **BOSS-05**: Boss has multi-phase AI with escalating attack patterns — Phase 4

### Tower System

- [x] **TOWR-01**: Each team has one Core Tower that deals high damage to nearby enemies — Phase 4
- [x] **TOWR-02**: Tower regenerates HP slowly when not under attack — Phase 4
- [x] **TOWR-03**: Tower is temporarily disabled when boss is killed — Phase 4
- [x] **TOWR-04**: Destroying the enemy Core Tower results in instant match victory — Phase 4

### Neutral Camps

- [x] **CAMP-01**: 4 neutral buff camps on the map: Damage, Shield, Haste, Cooldown — Phase 6
- [x] **CAMP-02**: Neutral camps respawn every 60 seconds after being cleared — Phase 6
- [x] **CAMP-03**: Killing a neutral camp grants a 30-second buff to the killing hero's team — Phase 6

### Draft

- [x] **DRFT-01**: Player is presented 3 random heroes and picks 1 before each match — Phase 8

### Ranked

- [x] **RANK-01**: Player progresses through rank tiers: Bronze, Silver, Gold, Platinum, Apex — Phase 8
- [x] **RANK-02**: MMR shifts +/-40 per match for fast climbing — Phase 8
